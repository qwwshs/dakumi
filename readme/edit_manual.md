# Dakumi 编辑器使用文档（Edit 界面）

## 界面布局

编辑界面分为三个主要区域：

| 区域 | 位置 | 功能 |
|------|------|------|
| **游玩区域（Demo）** | 左侧大区域 | 实时预览谱面，显示轨道、Note、判定线 |
| **编辑区域（Edit）** | 右侧竖条区域 | 分轨道编辑 Note 和 Event |
| **工具栏（EditTool）** | 顶部/侧边工具条 | 放置/切换工具、节拍、轨道、速度等控制 |

---

## 一、工具栏（EditTool）

工具栏位于编辑界面顶部，包含以下功能：

### 保存（Save）
- 快捷键：`Ctrl + S`
- 保存当前谱面到 `chart.json`
- 每 60 秒自动保存一次（可在设置中关闭）
- 退出时提示是否保存

### 播放控制（Music Play）
- 快捷键：`Space`
- 播放/暂停音乐
- 播放时会自动跟随节拍滚动

### 节拍（Denom）
- 快捷键：`↑`/`↓` 调整节拍分度
- 控制节拍线密度（1/4、1/8、1/16 等）
- 数字越大，可吸附的节拍越细

### 缩放（Scale）
- 控制编辑区域的纵向缩放比例
- 数值越大，显示的范围越大

### 轨道（Track）
- 快捷键：`←`/`→` 切换轨道
- 选择当前编辑的轨道编号
- 切换后编辑区域会显示对应轨道的 Note 和 Event

### 围栏（Fence）
- 设置轨道的围栏数量
- 用于辅助定位 Event 的 x 位置

### 播放速度（Speed）
- 控制音乐播放速度（0.1~）
- 不影响谱面，仅用于预览

### Note 假 Note 开关（Note Fake）
- 开启后放置的 Note 会被标记为假 Note
- 假 Note 在游玩中不显示判定，用于练习/装饰

### Hold 头部 Note 开关（Note Head）
- 开启后放置的 Hold Note 头部会附带一个 Tap Note
- 用于需要同时按住和点按的谱面

### Hold 头部 Wipe 开关（Wipe Head）
- 开启后放置的 Hold Note 头部会附带一个 Slide Note

---

## 二、游玩区域（Demo 区域）

### 轨道显示
- 显示所有轨道的实时位置（由 Event 驱动）
- 轨道编号和名称显示在判定线上方
- 当前选中的轨道会高亮显示
- 围栏线辅助对齐 Event 位置

### 判定线（Judge Line）
- 位于游玩区域中间
- Note 到达判定线时即为击打时机
- 进度条显示当前播放进度

### 实时信息
- 左上角显示：当前节拍（Beat）、时间（Time）
- 当前轨道的 x、w 值
- Note 总数和 Event 总数

### 轨道选择
- 在游玩区域鼠标左键点击轨道，可切换到该轨道
- 如果多轨道重叠，会自动切换选择

### 当前鼠标 x 位置
- 鼠标悬停在游玩区域时，显示当前 x 值和最近的围栏 x 值

---

## 三、编辑区域（Edit 区域）

编辑区域分为五个轨道栏目，从左到右依次为：

| 栏目 | 内容 |
|------|------|
| Note（音符） | 显示当前轨道的所有 Note |
| x（x 位置） | 轨道的 x 位置 Event |
| w（宽度） | 轨道的宽度 Event |
| lpos（左位置） | 轨道左边界位置 Event |
| rpos（右位置） | 轨道右边界位置 Event |

### Note 编辑
- **放置 Note**：快捷键 `Q`
- **放置 Wipe（滑动）**：快捷键 `W`
- **放置 Hold（长条）**：快捷键 `E`
  - 第一次按 E 放置头，第二次按 E 放置尾
- **删除**：快捷键 `D`，删除鼠标附近的 Note/Event
- **假 Note**：在工具栏打开假 Note 开关后放置

### Event 编辑
- **放置 Event**：快捷键 `E`（在 Event 轨道上）
  - 根据鼠标所在轨道自动选择 Event 类型（x/w/lpos/rpos）
  - 第一次点击确认头和 from 值，第二次点击确认尾和 to 值
- Event 支持两种过渡类型：
  - **Bezier（贝塞尔曲线）**：通过 4 个控制点定义变化曲线
  - **Easings（缓动函数）**：通过预定义的缓动函数定义变化曲线

### 框选（Select）
- 按住 `Shift` 拖拽鼠标进行框选
- 松开鼠标确认选择
- 选中后按 `Escape` 取消选择

### 快捷操作
- **右键点击** Note/Event：加入/取消选择
- **鼠标滚轮**：纵向滚动节拍
- **按住 Ctrl + 滚轮**：加速滚动

## 四、侧边栏（Sidebar）

侧边栏位于编辑区域右侧，显示详细信息和编辑面板。

### 功能入口
在侧边栏 `nil` 界面（未选择任何编辑模式时），点击按钮进入：

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| **Chart Info（谱面信息）** | 编辑曲名、谱师、艺术家、偏移量、BPM 列表 | - |
| **Preference（偏好）** | 调整 x 偏移、Event 缩放 | - |
| **Settings（设置）** | 全局设置 | - |
| **Track（轨道列表）** | 查看所有轨道的实时 x/w 值 | - |
| **Equalizer（均衡器）** | 音频均衡器调节 | - |
| **Takana（导出）** | 导出为 Takana 谱面格式 | - |
| **Dakumi** | 打开 Dakumi 官网 | - |
| **GitHub** | 打开 GitHub 项目页 | - |

### Chart Info（谱面信息）
- 编辑曲名、艺术家、谱师名称
- 调整偏移量（Offset，单位 ms）
- 管理 BPM 列表：
  - 点击 `Add` 在当前节拍位置添加 BPM 变化点
  - 点击 `Sub` 删除 BPM 变化点
  - 自动按节拍排序
- 点击提示按钮（勾）保存修改

### Preference（偏好设置）
- **x_offset（x 偏移）**：调整所有轨道的整体 x 偏移
- **event_scale（Event 缩放）**：调整 Event 数值的缩放比例
- 修改后点击提示按钮保存

### Settings（设置）
| 设置项 | 说明 |
|--------|------|
| Hit | 开启/关闭打击特效 |
| Hit Sound | 开启/关闭打击音效 |
| Hit Volume | 打击音效音量 |
| Hit Time | 打击特效持续时间 |
| Hit Light Time | 打击光效持续时间 |
| Wavfrom | 开启/关闭波形图显示 |
| Contact Roller | 鼠标滚动灵敏度 |
| Auto Save | 自动保存开关 |
| Default Trans Type | 默认过渡类型（easings/bezier） |
| Bg Alpha | 背景透明度 |
| Denom Alpha | 节拍线透明度 |
| Music Volume | 音乐音量 |
| Track W Scale | 轨道宽度倍数 |
| Note Height | Note 高度 |
| Judge Line Y | 判定线 Y 位置 |
| Angle | 3D 倾斜角度（预览效果） |
| Window Width/Height | 窗口尺寸 |
| Language | 语言设置 |
- 修改后点击提示按钮保存

### Track（轨道列表）
- 显示所有轨道的编号及当前 x、w 值
- 支持按 x 或 w 范围筛选
- 点击轨道编号切换到该轨道
- 点击 `Edit` 进入轨道属性编辑

### Track Edit（轨道编辑）
- **Track Name（轨道名称）**：为轨道设置名称
- **Do Not Hide（不隐藏）**：开启后即使 w=0 也显示轨道线
- **Parent（父轨道）**：设置父轨道编号，子轨道的 x 值会叠加父轨道 x
- 点击 `Break` 返回轨道列表

### Event（事件编辑）
选择了一个 Event 后进入该界面：
- **from / to**：编辑 Event 的起始值和结束值
- **同上/同下**：快速复制数值
- **Trans Type（过渡类型）**：切换 Bezier / Easings
- **Bezier 编辑**：
  - 通过逗号分隔的 4 个数值控制贝塞尔曲线
  - 可从预设贝塞尔列表中选取
  - 右侧显示实时曲线预览
- **Easings 编辑**：
  - 滑块选择缓动函数索引
  - 右侧显示实时曲线预览
- 修改后自动应用到选中的 Event

### Note（音符编辑）
选择了一个 Note 后进入该界面：
- **Fake（假 Note）**：设置 Note 是否为假 Note
- **Note Head（Hold 头部 Note）**：是否在 Hold 头部附带 Tap Note
- **Wipe Head（Hold 头部 Wipe）**：是否在 Hold 头部附带 Slide Note

### Events（批量事件编辑）
选中了多个 Event 后进入该界面：
- **快速调整**：上下按钮切换 Bezier/Easings 预设
- **执行（Do）**：对选中的 Event 批量执行数值变换
- **Perturbation（扰动）**：随机偏移量
- **From / To**：数值变化范围
- **Trans Expression（过渡表达式）**：
  - 支持 `bezier 1`、`easing 5`、`function x^2` 等表达式
  - 右侧显示实时曲线预览

## 五、快捷键汇总

### 基础编辑
| 按键 | 功能 |
|------|------|
| `Q` | 放置 Note |
| `W` | 放置 Wipe |
| `E` | 放置 Hold / Event |
| `D` | 删除鼠标附近的 Note/Event |
| `Tab` | 打开/关闭预览模式 |
| `Space` | 播放/暂停音乐 |
| `CapsLock` | 打开/关闭 Event 直观编辑 |

### 导航
| 按键 | 功能 |
|------|------|
| `←` | 轨道减 1 |
| `→` | 轨道加 1 |
| `↑` | 节拍分度加 1 |
| `↓` | 节拍分度减 1 |
| 滚轮 | 纵向滚动 |
| `Ctrl` + 滚轮 | 加速滚动 |

### 选区操作
| 按键 | 功能 |
|------|------|
| `Shift` + 拖拽 | 框选 |
| `Escape` | 取消选择 |
| 右键点击 | 单选/切换选择 |

### 剪贴板
| 按键 | 功能 |
|------|------|
| `Ctrl + C` | 复制选中内容 |
| `Ctrl + X` | 剪切选中内容 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + A + V` | 粘贴（包含 Event） |
| `Ctrl + B` | 翻转粘贴（Event 数值取反） |
| `Ctrl + A + B` | 翻转粘贴（包含 Event） |

### 删除选区
| 按键 | 功能 |
|------|------|
| `Ctrl + D` | 删除选中的 Note |
| `Ctrl + A + D` | 删除选中的 Note 和 Event |

### 撤销/重做
| 按键 | 功能 |
|------|------|
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |

### Alt 键组合（需选中 Note/Event 后使用）
| 按键 | 功能 |
|------|------|
| `Alt + Z` | 拖头（将 Note/Event 的头部拖到鼠标位置） |
| `Alt + X` | 拖尾（将 Note/Event 的尾部拖到鼠标位置） |
| `Alt + C` | 裁切 Event 为多个小段 |
| `Alt + B` | 翻转 Event 数值（镜像翻转 x 位置） |
| `Alt + T` | 快速调整 Event 数值（吸附到最近围栏） |
| `Alt + U` | 上下翻转 Event 的 from/to |

## 六、Event 直观编辑（CapsLock）

开启 `CapsLock` 后，选中的 Event 会在游玩区域显示可拖动的控制点：

- **头部圆点**：拖动修改 Event 的 from 值和节拍
- **尾部圆点**：拖动修改 Event 的 to 值和节拍
- **控制点（Bezier 模式）**：拖动调整贝塞尔曲线

右键点击控制点弹出菜单：
- 切换过渡类型
- 下一个/上一个预设曲线
- 添加/删除控制点

---

## 七、预览模式（Tab）

按 `Tab` 进入全屏预览模式，此时：
- 编辑界面隐藏
- 显示完整的游玩区域
- 可用于检查谱面效果

---

## 八、撤销/重做系统

- 所有对 Note/Event 的添加、删除都会自动记录到撤销栈
- `Ctrl + Z` 撤销上一步操作
- `Ctrl + Y` 重做被撤销的操作
- 注意：直接修改 Event 数值（如侧边栏编辑）不会记录到撤销栈

---

## 九、提示

1. **波形图**：在设置中开启后，编辑区域会显示音频波形，辅助对齐
2. **框选可跨轨道**：在游玩区域框选可以选中不同轨道的 Note
3. **轨道父子关系**：设置父轨道后，子轨道的 x 值会叠加父轨道，用于制作联动轨道
4. **围栏辅助**：设置围栏数量和 Event 直观编辑配合，可以精确对齐 Event 数值
5. **批量编辑**：选中多个 Event 后，在 Events 面板可以批量修改数值
